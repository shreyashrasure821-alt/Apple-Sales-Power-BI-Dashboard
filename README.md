APPLE SALES POWER BI DASHBOARD

Project Overview:
This project analyzes Apple retail sales data using Power BI
The Dashboard provides insights into project performance, catagory sales, store revenue, and yearly trends.

Key Features:
1. Revenue by Product
2. Revenue by Catagory
3. Revenue by Store
4. Yearly Revenue Trend
5. Interactive Slicers (Year and Catagory)

Revenue Calculation
Revenue=Price * Quantity

Tools Used
1. Power BI
2. Data Modeling
3. Data Visualization

Purpose:
This project demonstrates my ability to build interactive dashboards and perform sales data analysis using Power BI.

Dataset Information:
The dataset used in this project is a sample dataset created for learning and purposes. It does not represent actual Apple Inc. sales data.
